在Java中，所有的代码必须包含在某个类里。可以在类里写一个 public static void main(String[] args) （即main函数）函数，代码可以通过这里运行。另外，Java用分号和大括号分隔语句，没有缩进要求。最后，作为静态类型语言，Java中所有变量、方法等必须有一个已声明的类型，这个类型不能改变。
