数据类型与运算符

JavaScript有五种基本的数据类型：boolean、number、string、null、undefined。
数字运算符与其它语言相同，允许用加号进行字符串拼接。

比较运算符中，等于为 ===，不等为 !== 。双等号和不等号会自动类型转换后比较结果（比如 2 == ‘2’）。
