## Go语言入门

目标：用Go语言完成bookstore的部分功能

### 配置

直接官网下载。

解压到``/usr/local``文件夹：  

```bash
sudo tar -C /usr/local -xzf go.tar.gz
```

（``-C``用于制定目标目录，``-x``表示解压，``-z``使用gzip解压，``-f``指定归档（待解压）文件）

配置环境变量：

```bash
export GOROOT=/usr/local/go # 根目录
export GOPATH=$HOME/.go # 包下载目录
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin
```

``go version``检查配置。

### 语法

#### 初始化

```go
go mod init example/hello // 目录名
```

增加新的模块需求：

```go
go mod tidy
```

### 框架

Golang语言包含多个web框架，随便选了一个入门，使用Gin框架。

直接从github import。

```go
import "github.com/gin-gonic/gin"
```

#### controller

```go
package hello

import (
	"net/http"
	"github.com/gin-gonic/gin"
)

func Hello(c *gin.Context) {
	c.JSON(http.StatusOK, gin.H{
		"code": 200, "msg": "Hello World!",
	})
}
```

#### router

```go
package router

import (
	"github.com/gin-gonic/gin"
	"go-bookstore/controller/hello"
)

func StartRouter() *gin.Engine {
	router := gin.Default()
	
	router.GET("/hello", hello.Hello)

	return router
}
```
