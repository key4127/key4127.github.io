State Value​

记GtG_tGt​为某个轨迹的discounted return，则State Value（状态值）就是GtG_tGt​的期望值。

即

vπ(s)=E[Gt∣St=s]v_\pi(s) = E[G_t|S_t = s]vπ​(s)=E[Gt​∣St​=s]

状态值基于策略，是关于state的函数。

即为贝尔曼公式。

贝尔曼公式描述了不同状态值之间的关系。对于某个策略、状态等，贝尔曼公式实际是一组式子，联立可以求出各个状态量的具体值。

向量形式​

将贝尔曼公式写作

vπ(s)=rπ(s)+γΣs′pπ(s′∣s)vπ(s′)v_\pi(s) = r_\pi(s) + \gamma \mathop{\Sigma}\limits_{s\prime}p_\pi(s\prime|s)v_\pi(s\prime)vπ​(s)=rπ​(s)+γs′Σ​pπ​(s′∣s)vπ​(s′)

其中rπ(s)r_\pi(s)rπ​(s)是在现有策略下从sss出发立刻得到的平均奖励，pπ(s′∣s)p_\pi(s\prime|s)pπ​(s′∣s)是从sss到s′s\primes′的概率。

将sis_isi​从111到iii编号，则上式写为

vπ(si)=rπ(si)+γΣs′pπ(sj∣si)vπ(sj)v_\pi(s_i) = r_\pi(s_i) + \gamma \mathop{\Sigma}\limits_{s\prime}p_\pi(s_j|s_i)v_\pi(s_j)vπ​(si​)=rπ​(si​)+γs′Σ​pπ​(sj​∣si​)vπ​(sj​)

将所有式子排列在一起，即可写为矩阵形式

vπ=rπ+γPπvπv_\pi = r_\pi + \gamma P_\pi v_\pivπ​=rπ​+γPπ​vπ​

其中

  • vπ=[vπ(s1),...,vπ(sn)]Tv_\pi = [v_\pi(s_1), ..., v_\pi(s_n)]^Tvπ​=[vπ​(s1​),...,vπ​(sn​)]T
  • rπ=[rπ(s1),...,rπ(sn)]Tr_\pi = [r_\pi(s_1), ..., r_\pi(s_n)]^Trπ​=[rπ​(s1​),...,rπ​(sn​)]T
  • Pπ∈Rn×n,[Pπ]ij=pπ(sj∣si)P_\pi \in R^{n \times n}, [P_\pi]_{ij} = p_\pi(s_j|s_i)Pπ​∈Rn×n,[Pπ​]ij​=pπ​(sj​∣si​)，PπP_\piPπ​为状态转移矩阵

最简单（但复杂度高）的求解方式是求逆。

vπ=(I−γPπ)−1rπv_\pi = (I - \gamma P_\pi)^{-1}r_\pivπ​=(I−γPπ​)−1rπ​

另一种方式是迭代。可以假设其中某个viv_ivi​是任一一个值，将其带入求vi+1v_{i + 1}vi+1​，不断迭代，当迭代次数趋于无穷时，对应的状态值趋于真实值。

Action Value​

action value（动作价值）指在某个状态下采取某个行动，能得到的平均回报。action value用于衡量在某个状态下，哪个动作的价值更高。

即有

qπ(s,a)=E[Gt∣St=s,At=a]q_\pi(s, a) = E[G_t|S_t = s, A_t = a]qπ​(s,a)=E[Gt​∣St​=s,At​=a] vπ(s)=Σaπ(a∣s)qπ(s,a)v_\pi(s) = \mathop{\Sigma}\limits_{a}\pi(a|s)q_\pi(s,a) vπ​(s)=aΣ​π(a∣s)qπ​(s,a)

和贝尔曼公式联立，有

qπ(s,a)=Σrp(r∣s,a)r+γΣs′p(s′∣s,a)vπ(s′)q_\pi(s,a) = \mathop{\Sigma}\limits_{r}p(r|s, a)r + \gamma \mathop{\Sigma}\limits_{s\prime}p(s\prime|s,a)v_\pi(s\prime)qπ​(s,a)=rΣ​p(r∣s,a)r+γs′Σ​p(s′∣s,a)vπ​(s′)

这个式子说明状态价值和动作价值可以相互转化。
