03 Deduction

这篇笔记包含lecture7，Deduction ppt 中的内容。上一个ppt（The Application of Propositional Logic）介绍了SAT的具体使用，跳过。

对应教材《数理逻辑与集合论》中1.6、2.5、2.7~2.10。

Deduction​

现有两个公式 AAA 和 BBB ，如果 AAA 为真时 BBB 必定为真，则 AAA 重言蕴含 BBB （A  ⟹  BA \implies BA⟹B）。

相关公式：

  • If A  ⟹  BA \implies BA⟹B, A is a tautology, then B is a tautology.
  • If A  ⟹  BA \implies BA⟹B, B  ⟹  AB \implies AB⟹A, then A=BA = BA=B.
  • If A  ⟹  BA \implies BA⟹B, B  ⟹  CB \implies CB⟹C, then A  ⟹  CA \implies CA⟹C.
  • If A  ⟹  BA \implies BA⟹B, A  ⟹  CA \implies CA⟹C, then A  ⟹  B∧CA \implies B \land CA⟹B∧C.
  • If A  ⟹  CA \implies CA⟹C, B  ⟹  CB \implies CB⟹C, then A∨B  ⟹  CA \lor B \implies CA∨B⟹C.

相关规则：

  • 代入规则（Substitution Rule）：对于永真式 AAA 中的任何命题变量 PPP ，用另一个公式替换 AAA 中的所有 PPP 得到新公式 BBB ，则 BBB 也是永真式。
  • 置换规则（Replacement Rule）：对于公式 AAA 中的某个命题变量 PPP ，将其全部替换为某个公式得到新公式 BBB ，两公式 AAA 和 BBB 具有相同的真值表。
  • 条件证明规则：A∨B  ⟹  CA \lor B \implies CA∨B⟹C iff A  ⟹  B→CA \implies B \rightarrow CA⟹B→C 。

对偶式​

对于公式 AAA ，将 AAA 中的 ∨\lor∨ 、 ∧\land∧ 、 TTT 和 FFF 对应换为 ∧\land∧ 、 ∨\lor∨ 、 FFF 和 TTT ，得到公式 A∗A^*A∗ ，则 AAA 和 A∗A^*A∗ 互为对偶式。例如， (P∧Q)∨T(P \land Q) \lor T(P∧Q)∨T 和 (P∨Q)∧F(P \lor Q) \land F(P∨Q)∧F 互为对偶式。

Polish Notation​

前文中的WFF是中缀表达式。相应的，还有前缀式（波兰式）和后缀式（逆波兰式），与这里的转换普通前中后缀转换相同。

简单来说，对于一个WFF，将其写成二叉树的形式。则这棵树的前序遍历、中序遍历、后序遍历分别是前中后缀表达式。
