命题​

命题及相关基本定义​

命题的定义是 “a statement that is, by itself, either true or false”。由此，不确定真假的、感叹句、祈使句、问句不是命题。命题逻辑的定义是 “a mathematical system for reasoning about propositions and how they relate to one another”。

命题逻辑公式由命题变项和命题联结词组成。命题变项代表命题，命题联结词代表命题之间的关系。命题变项有以下几个特点：一个命题变项代表一个简单命题（原子命题），用P、Q等大写字母表示、要么为真要么为假。命题联结词用于连接命题。

此外，没有联结词的命题被称简单命题（原子命题）。

合式公式（WFF）​

如果一个公式是 “a well-formed formula”，则这个公式是合式公式（WFF）。

合式公式的递归定义如下：

  1. 每个单独的命题是WFF。
  2. 如果A、B是WFF，则 ¬A\neg A¬A 、 A∧BA \land BA∧B 、 A∨BA \lor BA∨B 、 A→BA \rightarrow BA→B 、 A↔BA \leftrightarrow BA↔B 是WFF。
  3. 除1、2之外的任何公式都不是WFF。

联结词​

联结词完备性​

如果一组联结词满足下面的条件，则它们是完备的：

  • 每个公式都可以转换成等价公式，并且这个等价公式只使用组里的联结词。

而一个有n个变量的公式可以看作一个有n个参数的函数。如果每个这样的函数都可以转为WFF，则联结词是完备的。而如果某个函数的真值表和公式相同，则它们是等价的。

如何将真值表转换为公式？以下面为例：

PPPQQQg1(P,Q)g1(P, Q)g1(P,Q)
TTTTTTTTT
TTTFFFTTT
FFFTTTFFF
FFFFFFFFF

  • 考虑结果为T的行

    • 选出为T的每一行，析取P、Q原值：
      P∧QP \land QP∧Q      P∧¬QP \land \neg QP∧¬Q     

    • 用 ∨\lor∨ 连接：
      (P∧Q)∨(P∧¬Q)(P \land Q) \lor (P \land \neg Q)(P∧Q)∨(P∧¬Q)

  • 考虑结果为F的行

    • 选出为F的每一行，合取P、Q取反的值（只要有一个为真结果就不是F）：
      ¬P∧¬Q\neg P \land \neg Q¬P∧¬Q      ¬P∧Q\neg P \land Q¬P∧Q     

    • 用 ∧\land∧ 连接：
      (¬P∧¬Q)∧(¬P∧Q)(\neg P \land \neg Q) \land (\neg P \land Q)(¬P∧¬Q)∧(¬P∧Q)

对于现有的联结词，已经知道

P↔Q=(P→Q)∧(Q→P)P \leftrightarrow Q = (P \rightarrow Q) \land (Q \rightarrow P)P↔Q=(P→Q)∧(Q→P)

则 {¬\neg¬、∧\land∧、∨\lor∨、→\rightarrow→} 是完备的。

更近一步，可以发现

P→Q=(¬P)∨QP \rightarrow Q = (\neg P) \lor QP→Q=(¬P)∨Q

由摩根定律得

P∨Q=¬((¬P)∧(¬Q))P \lor Q = \neg ((\neg P) \land (\neg Q))P∨Q=¬((¬P)∧(¬Q))
P∧Q=¬((¬P)∨(¬Q))P \land Q = \neg ((\neg P) \lor (\neg Q))P∧Q=¬((¬P)∨(¬Q))

则 {¬\neg¬, ∧\land∧}，{¬\neg¬, ∨\lor∨} 是完备的。

相关定律​

如果两个公式的真值表相同，则它们是等值的。

等值定理
P=QP = QP=Q iff P↔QP \leftrightarrow QP↔Q is always true.

摩根定理
¬(P∧Q)=¬P∨¬Q\neg (P \land Q) = \neg P \lor \neg Q¬(P∧Q)=¬P∨¬Q ¬(P∨Q)=¬P∧¬Q\neg (P \lor Q) = \neg P \land \neg Q¬(P∨Q)=¬P∧¬Q

双重否定律
¬¬P=P\neg \neg P = P¬¬P=P

结合律
(P∧Q)∧R=P∧(Q∧R)(P \land Q) \land R = P \land (Q \land R)(P∧Q)∧R=P∧(Q∧R)
(P∨Q)∨R=P∨(Q∨R)(P \lor Q) \lor R = P \lor (Q \lor R)(P∨Q)∨R=P∨(Q∨R)
(P↔Q)↔R=P↔(Q↔R)(P \leftrightarrow Q) \leftrightarrow R = P \leftrightarrow (Q \leftrightarrow R)(P↔Q)↔R=P↔(Q↔R)
（对 →\rightarrow→ 不适用）

交换律
P∨Q=Q∨PP \lor Q = Q \lor PP∨Q=Q∨P
P∧Q=Q∧PP \land Q = Q \land PP∧Q=Q∧P
P↔Q=Q↔PP \leftrightarrow Q = Q \leftrightarrow PP↔Q=Q↔P
（对 →\rightarrow→ 不适用）

等幂律
P∧P=PP \land P = PP∧P=P
P∨P=PP \lor P = PP∨P=P
P→P=TP \rightarrow P = TP→P=T
P↔P=TP \leftrightarrow P = TP↔P=T

同一律
P∧T=PP \land T = PP∧T=P
P∨F=PP \lor F = PP∨F=P
T→P=PT \rightarrow P = PT→P=P
T↔P=PT \leftrightarrow P = PT↔P=P
P→F=¬PP \rightarrow F = \neg PP→F=¬P
P↔F=¬PP \leftrightarrow F = \neg PP↔F=¬P

补余律
P∧¬P=FP \land \neg P = FP∧¬P=F
P∨¬P=TP \lor \neg P = TP∨¬P=T
P→¬P=¬PP \rightarrow \neg P = \neg PP→¬P=¬P
¬P→P=P\neg P \rightarrow P = P¬P→P=P
P↔¬P=FP \leftrightarrow \neg P = FP↔¬P=F

零律
P∧F=FP \land F = FP∧F=F
P∨T=TP \lor T = TP∨T=T
P→T=TP \rightarrow T = TP→T=T
F→P=TF \rightarrow P = TF→P=T

分配律
P∨(Q1∧Q2∧...∧Qn)=(P∨Q1)∧(P∨Q2)∧...∧(P∨Qn)P \lor (Q_1 \land Q_2 \land ... \land Q_n) = (P \lor Q_1) \land (P \lor Q_2) \land ... \land (P \lor Q_n)P∨(Q1​∧Q2​∧...∧Qn​)=(P∨Q1​)∧(P∨Q2​)∧...∧(P∨Qn​)

P∨(P∧Q)=PP \lor (P \land Q) = PP∨(P∧Q)=P
P∧(P∨Q)=PP \land (P \lor Q) = PP∧(P∨Q)=P
