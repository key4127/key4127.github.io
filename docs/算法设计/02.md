## 分治

主定理：

如果$T(n) = aT(n/b) + O(n^d)$，则

$T(n)=
\begin{cases}
O(n^d),~if~d > log_ba \\
O(n^d logn),~if~d = log_ba \\
O(n^{log_ba}),~if~d < log_ba
\end{cases}$

Akra–Bazzi定理是主定理的推广。令$|h_i(n)|=O(n/log^2n)$，$g(n)=O(n^c)$，且

$
T(n) = \Sigma a_iT(b_in+h_i(n)) + g(n)
$

则$T(n) = \Theta(n^p(1 + \int_1^n g(u) / (u^p + 1)du))$，其中$p$满足$\Sigma a_ib_i^p = 1$

排序算法可以写为树，每种排列对应一个节点。则基于比较的时间复杂度是$log_2(n!)$，根据斯特林公式，比较排序算法最坏情况下的时间复杂度下界是$\Omega(nlogn)$。

中位数选取时，可以每次随机选择主元，用类似快排的方式分治。这种方法的平均复杂度为$O(n)$。
